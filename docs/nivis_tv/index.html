How to stream to nivis_tv

  1. Setup your own twitch channel, then simply type !host yourTwitchUsername in the FC Discord server for nivis_tv to host you.
  2. If you will only stream FFXIV and no other games, you can ask Yuki to be added to the autohost.
  3. If you don't want to setup your own channel, and would like to stream to nivis_tv directly (i.e. not hosted), then ask Yuki for the stream key.
