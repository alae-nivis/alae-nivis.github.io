FC's Choice Event

Vote for the event you want to see!

The FC's Choice Event runs on the first content (non-treasure hunt) weekend of every month, but the specific time and date (Saturday or Sunday) is voted on.

Current voting form can be found here:

https://forms.gle/QcEb1LNQQi3fcZpq6

For the next scheduled event please check the Events page.

Example event options can be found below. If the event you want is not on the voting form then there is an option available for you to write it and it will be included in the next month's voting form. The events are not limited to the list below. The FC currently has over 200 million gil in cash that is sitting there unused. Some of it can go to event rewards.

To control the number of options in each vote the least popular option may or may not be removed from future votes. It can be added back in from a write-in option later. This is to prevent voting forms from looking thusly.
