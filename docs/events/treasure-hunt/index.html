Unlocking

Treasure Hunts

Check if you have the Decipher and Dig skills under Actions & Traits -> General. If you do, you're set to go.

If not, complete the Treasures and Tribulations quest from NPC: H'loonh at Eastern La Noscea (x21, y21).

Flying

Flying is required in the zones where maps are. 30 zones total, 6 per expansion. Check your Aether Currents menu if you are unsure. It should say Attunement Complete on all of them.

If you are over level 60 and unsure about how to fly / how to unlock flying, contact Yuki immediately.
