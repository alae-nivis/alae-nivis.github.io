Unlocking

Treasure Hunts

Check if you have the Decipher and Dig skills under Actions & Traits -> General. If you do, you're set to go.

If not, complete the Treasures and Tribulations quest from NPC: H'loonh at Eastern La Noscea (x21, y21).

Flying

Flying is required in the zones where maps are. 24 zones total, 6 per expansion. Check your Aether Currents menu if you are unsure. It should say Attunement Complete on all of them.

If you are level 90 and unsure about how to fly / how to unlock flying, contact Yuki immediately.

Treasure Maps

We are running portal spawning maps. This can be Dragonskin (lv60), Gazelleskin (lv70), Zonureskin (lv80), or Kumbhiraskin (lv90). Any other grade map you can complete it solo (that means you get all the loot). If a Timeworn Thief's Map drops we will do that immediately after the current map.

You can gather maps yourself with botanist or miner if you have one levelled. Alternatively you can buy maps from the market board for around 40k gil.
